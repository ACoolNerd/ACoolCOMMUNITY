# Contributing to ACoolECOSYSTEM

## Standards
- **Branching**: Use `feature/ACoolFEATURE-name` or `bugfix/ACoolBUG-name`.
- **Commits**: Follow `[ACoolTYPE] action: description` format.
- **PRs**: Must pass all CI/CD stages and governance checks.

## Workflow
1. Fork the template.
2. Create your branch.
3. Commit changes.
4. Open a Merge Request to `main`.
